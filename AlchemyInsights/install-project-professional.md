# <a name="install-project-professional"></a>Instalați Project Professional

Consultați [Instalarea Project](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) pentru a afla cum să instalați versiunea cu abonament de Project Online și versiunea fără abonament de Project 2019, 2016 sau 2013. 

Pentru Project 2010 sau 2007, urmați pașii din [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) [sau Office 2007.](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657) 

Project se vinde ca aplicație stand-alone, așadar, dacă încă nu o aveți, cumpărați sau [încercați Project](https://www.microsoft.com/evalcenter/evaluate-project).
