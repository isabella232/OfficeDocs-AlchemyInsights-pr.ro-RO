# <a name="reduce-the-size-of-your-outlook-data-file"></a>Reducerea dimensiunii fișierului Outlook date

Pentru a reduce dimensiunea fișierului de date Outlook (.pst) sau fișierului de date offline Outlook (.ost), puteți compacta manual fișierul de date. 

Pentru instrucțiuni despre cum să reduceți dimensiunea unui fișier .pst sau .ost și să selectați versiunea de Outlook, consultați Reducerea dimensiunii cutiei poștale și a fișierelor de date [Outlook (.pst și .ost).](https://support.microsoft.com/office/reduce-the-size-of-your-mailbox-and-outlook-data-files-pst-and-ost-e4c6a4f1-d39c-47dc-a4fa-abe96dc8c7ef)
