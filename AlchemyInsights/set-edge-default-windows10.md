# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Setați Microsoft Edge ca browser implicit pe un dispozitiv Windows 10

Iată cum se procedează:

1. Selectați **butonul Start,** apoi introduceți **Aplicații implicite**.

1. În rezultatele căutării, selectați **Aplicații implicite**.

1. Sub **Browser web,** selectați **Microsoft Edge**.
