# <a name="create-a-folder-in-outlookcom"></a>Crearea unui folder în Outlook.com

1. În partea de jos a panoului de foldere, **selectați Folder nou**. Poate fi necesar să defilați în jos în panoul de foldere pentru a vedea **butonul Folder** nou.
2. În caseta text, introduceți un nume pentru folder, apoi apăsați pe Enter.

Pentru a afla mai multe, [consultați Crearea unui folder în Outlook.com.](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)
