# <a name="help-signing-in-with-windows-hello-pin"></a>Ajutor la conectarea cu pinul Windows Hello

- Pentru a schimba, a reinițializa sau a recupera codul PIN (atunci când vă conectat), accesați Schimbați codul PIN atunci când [ați fost conectat deja la dispozitiv.](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19)

- Pentru a schimba, a reseta sau a recupera codul PIN (atunci când sunteți conectat), accesați Resetați codul PIN atunci când nu [sunteți conectat la Windows 10.](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904)

- Pentru a configura codul PIN Windows Hello, vizitați [Aflați mai multe despre Windows Hello și configurați-l.](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0)
