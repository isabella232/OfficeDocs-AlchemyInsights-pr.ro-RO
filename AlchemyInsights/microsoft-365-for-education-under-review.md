# <a name="microsoft-365-for-education---under-review"></a>Microsoft 365 Education - în curs de revizuire

Eligibilitatea dvs. ca Institut academic calificat pentru Microsoft este **în curs de revizuire**. Vă puteți aștepta să primiți o notificare privind stabilirea eligibilității cu **48 de ore** înainte de ora remiterii. Acordați cel puțin **48 de ore** pentru a primi această notificare înainte de a deschide o solicitare de serviciu.

Pentru mai multe informații, consultați [Microsoft 365 Education](https://www.microsoft.com/education/buy-license/microsoft365).
