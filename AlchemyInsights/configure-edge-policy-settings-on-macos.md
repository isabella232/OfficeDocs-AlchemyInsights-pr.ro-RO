# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>UtilizațiMgf pentru a configura setările de politică Microsoft Edge pe macOS

Pentru a configura Microsoft Edge (canalul stabil 81 sau o versiune mai recentă) pe macOS, utilizați un fișier manifest de politică Microsoft Edge pe Xmlf Pro 10.19.

Pentru mai multe informații, [consultați Configurarea setărilor de politică Microsoft Edge pe macOS cu Faceți clic pe](https://go.microsoft.com/fwlink/?linkid=2134761).
