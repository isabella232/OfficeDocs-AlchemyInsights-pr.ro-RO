# <a name="single-sign-on-sso-in-microsoft-edge"></a>Sign-on unic (SSO) în Microsoft Edge

Microsoft Edge acceptă următoarele tipuri de sign-on unic (SSO):
- SSO bazat pe PRT
- SSO perfect
- SSO bazat pe WIA
