# <a name="yammer-web-parts-in-sharepoint"></a>Yammer web din SharePoint

Yammer Conversații și Yammer Evidențiază părțile web îmbunătățesc colaborarea pe paginile SharePoint moderne și clasice. Pentru mai multe informații, consultați [Yammer Despre Conversații](https://support.microsoft.com/office/use-a-yammer-web-part-in-sharepoint-online-a53cfa0c-3d09-42c8-a286-1038a81c59da#conversations) [Yammer Highlight-uri](https://support.microsoft.com/office/use-a-yammer-web-part-in-sharepoint-online-a53cfa0c-3d09-42c8-a286-1038a81c59da#highlights).    

Partea web Yammer Conversații cu sistem de nume modern este actualizată la noua experiență Yammer nou și este disponibilă pentru entitățile găzduite Lansare pentru țintit. Rulaj GA a început. Caracteristicile noi includ posibilitatea de a începe o conversație cu orice postare (Întrebări, Sondaje, Laude) și de a marca cele mai bune răspunsuri direct din partea SharePoint. Pentru mai multe informații, consultați [Termeni Yammer pentru clienți și Întrebări frecvente.](https://docs.microsoft.com/yammer/get-started-with-yammer/newyammer-faq)

 Pentru a înțelege ce parte web și configurare sunt potrivită pentru dvs., consultați Utilizarea [unei Yammer web SharePoint Online.](https://support.microsoft.com/office/use-a-yammer-web-part-in-sharepoint-online-a53cfa0c-3d09-42c8-a286-1038a81c59da)  

**Utilizarea părților web cu SharePoint Server**  

Părțile web pot fi utilizate în pagini moderne și clasice în mediile local.

- Pentru mai multe informații despre paginile moderne, [consultați Adăugarea unui flux Yammer la o pagină modernă din SharePoint Server 2019](https://docs.microsoft.com/yammer/integrate-yammer-with-other-apps/embed-a-feed-into-a-sharepoint-site#add-a-yammer-feed-to-a-modern-page-in-sharepoint-server-2019). 
- Pentru mai multe informații despre paginile clasice, consultați Adăugarea unui flux Yammer la o pagină clasică în [SharePoint Servers 2013, 2016 și 2019.](https://docs.microsoft.com/yammer/integrate-yammer-with-other-apps/embed-a-feed-into-a-sharepoint-site#add-a-yammer-feed-to-a-classic-page-in-sharepoint-servers-2013-2016-and-2019)

**Yammer Încorporare**  

Utilizați instrumentul Configurație încorporată pentru a testa utilizarea încorporarii. O actualizare viitoare la Încorporare va activa noua Yammer încorporată. Pentru mai multe informații, consultați Instrumentul [Yammer Configurare încorporată.](https://aka.ms/YammerEmbedConfigureTool) Pentru a înțelege mai bine Yammer încorporată, consultați [Încorporare flux.](https://aka.ms/YammerDevDocs)

**Probleme și limite cunoscute**

- ID-urile de grup nu sunt disponibile din noile URL-Yammer de grup, care s-au modificat. Comutați înapoi la modul clasic pentru a obține ID-uri de grup sau alte URL-uri de la URL-uri.
- Domeniile particularizate (autoturism) nu sunt acceptate.
- Yammer Încorporarea nu este optimizată pentru dispozitive mobile. Utilizați pagini moderne cu partea Yammer web Conversații, pentru cea mai bună experiență.
- Temele particularizate pot afecta aspectul și modul de utilizare ale Yammer Web Conversații. Deschideți un caz de asistență pentru a raporta probleme.
