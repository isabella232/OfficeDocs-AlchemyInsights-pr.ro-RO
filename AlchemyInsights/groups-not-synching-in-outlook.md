# <a name="microsoft-365-groups-not-synching-in-outlook"></a>Microsoft 365 Grupurile nu se s synching în Outlook

Pentru a Microsoft 365 probleme de sincronizare a e-mailului din grup, trebuie să activați OAuth (autentificare modernă). 

Pentru pașii de activare a OAuth, consultați [Activarea sau dezactivarea autentificării moderne pentru utilizatorii Outlook în Exchange Online](/exchange/clients-and-mobile-in-exchange-online/enable-or-disable-modern-authentication-in-exchange-online).
